Note: The Shapiro-Wilk test is a formal test for normality that's more reliable than visual assessments like Q-Q plots. While Q-Q plots can sometimes give the illusion of normality, the Shapiro-Wilk test provides a quantitative, objective measure of how well the data follows normal distribution. We will quickly explore this data's normality, in contrast to the QQ plot above.

- **Test Statistic:** 0.987
- **P-value:** 0.0023

**Interpretation:** With a p-value < 0.05, we reject the null hypothesis that the data follows a normal distribution, indicating that non-parametric methods may be more appropriate for this analysis.

## Model Selection Process

To determine the optimal regression model for predicting sleep quality, we systematically evaluated various predictor combinations using both statistical criteria and model diagnostics. The selection process involved the following steps:

### 1. Initial Screening of Predictors

All potential variables were initially considered:

**Demographics:** Age, Gender, Occupation

**Lifestyle:** Physical Activity Level, Stress Level, Daily Steps, Heart Rate

**Health:** BMI Category

We first used correlation analysis and domain understanding to filter variables that lacked theoretical or statistical relevance.

### 2. Multicollinearity Check

Using statsmodels' variance_inflation_factor, we computed VIF scores for each predictor in the initial model. Variables with VIF > 5 were flagged for potential multicollinearity. To resolve this:

- Highly collinear variables were removed
- In some cases, domain relevance overruled VIF thresholds if interpretability was prioritized

### 3. Stepwise Regression with AIC

We implemented stepwise regression (both forward and backward) using statsmodels and a custom loop that minimizes Akaike Information Criterion (AIC).

**Procedure:**
- Start with an intercept-only model
- Iteratively add or remove predictors based on which change yields the largest AIC reduction
- Continue until no further improvement is possible

This method balances model complexity and fit and helps avoid overfitting.

### 4. Cross-Validation for Generalizability

To validate predictive performance, we used 5-fold cross-validation with scikit-learn's cross_val_score on a linear regression model (LinearRegression()):

- Evaluated using negative mean squared error (neg-MSE) and RÂ²
- Averaged scores across folds for fair comparison between models

This ensured our final model generalized well beyond the training set.

### 5. Final Model Selection

The best model based on AIC and cross-validation performance included the following predictors:

- Age
- Physical Activity Level
- Stress Level
- Heart Rate
- Daily Steps

These variables consistently showed strong predictive power, low multicollinearity, and satisfied key model assumptions.

### 6. Final Diagnostics

After selecting the model, we validated its assumptions:

- **Linearity:** Verified via residuals vs fitted plots
- **Normality of Residuals:** Checked using Q-Q plots and the Shapiro-Wilk test
- **Homoscedasticity:** Assessed via Breusch-Pagan test and residual spread
- **Independence:** No time-based autocorrelation present
- **Multicollinearity:** Final VIF values all < 5
